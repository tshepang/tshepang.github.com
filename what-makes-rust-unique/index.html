what makes Rust unique

2015-01-24

An interesting expansion of "safety without garbage collection":

It's easy to write (a) an unsafe language that has no GC, or (b) a safe language that relies on GC for safety. It's also easy to write a language that is in category (b) but lets you drop down to category (a) in an unsafe dialect. It's a very difficult problem to write a safe language that does not rely on GC, and I know of no other industry language that has done it.

Part of a response by a member of Rust core team.
