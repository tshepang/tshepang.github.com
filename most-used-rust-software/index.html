most used Rust software

This is in the sense of largest number of end users

  • millions use TiKV (adopters), a distributed transactional key-value database. One of those adopters has over 200m users (mid 2018). This is, for me, the most impressive Rust success story.

  • 500m people use Dropbox (early 2016), whose file storage system is written in Rus.

  • 250m people use Firefox (late 2019), which has various components written in Rust, most notably Stylo, which does CSS style calculation.

  • 250m people use Discord (early 2020), which has Read States service written in Rust, whose "sole purpose is to keep track of which channels and messages you have read".
